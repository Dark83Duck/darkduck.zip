[0단계 : 구글 계정 로그인]									
이 프로그램을 사용하기 위해서는 반드시 구글 계정에 로그인 해야 합니다.

[1단계 : 시트 자동 준비하기]									
① 먼저, 구글 시트 메뉴의 가장 오른쪽의 [🍎 초등 생활지도 도구] 를 클릭합니다.									
② [1단계: 시트 자동 준비하기] 를 클릭합니다.									
③ [승인 필요] 팝업에서 [확인]을 클릭합니다.									
④ 왼쪽 하단의 [고급]을 클릭합니다.									
⑤ [딸깍학급메모(으)로 이동(안전하지 않음)]을 클릭합니다.									
⑥ 체크박스를 모두 체크하고 [계속]을 클릭합니다.									
⑦ 구글 시트 아래에 [명단], [기록]이라는 시트가 자동 생성됩니다.									
⑧ [명단]시트로 이동하여 학급 학생의 이름을 입력합니다.									
									
[1-1단계 : 웹앱 배포하기									
① 구글 시트의 상단 메뉴에서 [확장프로그램] - [Apps Script]를 클릭합니다.									
② Apps Script의 상단 오른쪽 메뉴에서 [배포] - [새배포]를 클릭합니다.									
③ 유형 선택 오른쪽의 톱니바퀴 - [웹 앱]을 클릭합니다.									
④ 하단의 [배포]를 클릭합니다.									
⑤ [엑세스 승인]을 클릭합니다.									
⑥ 왼쪽 하단의 [Advanced]를 클릭합니다.									
⑦ [Go to 딸깍학급메모 (unsafe)]를 클릭합니다.									
⑧ 체크박스를 모두 체크하고 [Continue]를 클릭합니다.									
⑨ 웹 앱의 URL을 복사(클릭)한 후, [완료]를 클릭합니다.									
									
[2단계 : 웹앱 화면 열기									
① 구글 시트 메뉴의 가장 오른쪽의 [🍎 초등 생활지도 도구] 를 클릭합니다.									
② [2단계: 웹앱 주소 등록하기] 를 클릭합니다.									
③ 복사했던 URL을 붙여넣고 [확인]을 클릭합니다.									
									
[3단계 : 웹앱 화면 열기									
① 구글 시트 메뉴의 가장 오른쪽의 [🍎 초등 생활지도 도구] 를 클릭합니다.									
② [3단계: 웹앱 화면 열기] 를 클릭합니다.									
③ [바로 열기]를 클릭하여 실행하거나, [주소 복사]하여 브라우져의 즐겨찾기에 등록합니다.
