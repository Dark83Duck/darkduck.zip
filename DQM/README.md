던전 퀴즈 마스터 사용방법

1. 구글 드라이브에서 Google 스프레드시트를 만듭니다.

2. 구글 시트의 A1에는 문제, B1에는 보기1, C1에는 보기2, D1에는 보기3, E1에는 보기4, F1에는 정답 이라고 입력합니다.

3. 2열부터 출제하고 싶은 문제와 보기1~보기4까지 입력하고 정답에는 숫자만 입력합니다. (예를 들어 보기2가 정답이면 2를 입력)

4. 오른쪽 상단의 공유버튼을 클릭하여 일반엑세스에서 「링크가 있는 모든 사용자」로 설정은 변경합니다.

5. 역할은 「뷰어」로 변경합니다.

6. 링크를 복사합니다.

7. https://던전퀴즈마스터.qaa.kr 또는 https://dark83duck.github.io/RDQuiz/로 접속합니다.

8. 던전 지도(구글 시트 링크)란에 복사한 구글 시트 링크를 붙여넣습니다.

9. 정상작동하는지 보려면 「던전 입장하기」를, 학생들을 참여시키려면 「소환 마법진(QR 코드 생성)」을 선택합니다.
